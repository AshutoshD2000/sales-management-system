# Sales Management System

## 📌 Description
This is a Django-based Sales Management System that helps manage sales records efficiently.

## 🚀 Features
- Add Sales
- Update Sales
- Delete Sales
- View Sales List

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS

## ▶️ How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. python manage.py runserver
